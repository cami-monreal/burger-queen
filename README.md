# Buda Vegan Burger
Sidarta (Buda), propagou seus conhecimentos sobre a terra, ele quer acabar com o sofrimento de qualquer espécie. 
Nossa lanchonete e Cafeteria é vegana, evitando sofrimento animal e trazendo alegria.
